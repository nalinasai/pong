**Pong Game – Python Turtle
Overview**

This project is a classic two-player Pong game built using Python’s Turtle graphics module. Two paddles are controlled via the keyboard, and players compete to score points by bouncing the ball past the opponent’s paddle.

**Technologies Used**

Python 3

Turtle Graphics

Keyboard Event Handling

**Gameplay**

Player A controls the left paddle

Player B controls the right paddle

The ball moves continuously and bounces off walls and paddles

A player scores when the opponent fails to hit the ball

Scores are displayed at the top of the window
